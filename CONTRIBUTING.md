# Contributing To StreamHub Deck

StreamHub Deck is focused on SteamOS, Steam Deck Gaming Mode, streaming-platform launch behavior, and IPTV usability.

## Useful Areas

- Steam Deck controller navigation
- Steam Input shortcut behavior
- IPTV playback compatibility
- Xtream Codes and M3U profile handling
- Linux packaging and installer reliability
- Steam library artwork and UI polish

## Development

```bash
npm install
npm run dev
npm run check
```

Before proposing a change, verify that the app still works with keyboard input, physical directional navigation, IPTV login fields, and platform menu shortcuts.

## Release Notes

Each release should include clear notes describing user-facing changes, validation performed, and any known SteamOS limitations.
