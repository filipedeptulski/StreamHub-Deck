StreamHub Deck

A SteamOS streaming launcher for Steam Deck Gaming Mode, persistent platform profiles, and IPTV playback with Xtream Codes and M3U support.

Download latest release View on GitHub

Streaming Platforms

Launch Netflix, Disney+, Max, Crunchyroll, Prime Video, YouTube, Apple TV+, Paramount+, Hulu, Peacock, Plex, Tubi, Pluto TV, Roku, Rumble, ESPN and more from one Steam Deck interface.

IPTV

Use Xtream Codes API or M3U URL profiles with live TV, movies, series, channel categories, account expiration display, HLS and MPEGTS compatibility.

Steam Deck Controls

Built around SteamOS Gaming Mode navigation, physical controls, F9 app menu shortcut, and browser back/forward subcommands.

Automatic Updates

StreamHub Deck checks GitHub Releases, verifies SHA256 metadata, and installs new SteamOS packages when the user confirms.

Install

Download the latest StreamHub-Deck-vX.X.X-x64.tar.gz from GitHub Releases, extract it in ~/Downloads, and run ./install-streamhub-deck.sh.
